# projectSOL

Aplicacion movil de pronostico solar que predice cuantos volts/kWh generaran los paneles solares de un usuario basado en el clima (nubes y temperatura) de su ubicacion actual. La app ayuda al usuario a decidir a que hora del dia conectar sus electrodomesticos para ahorrar energia.

## Caracteristicas Principales

- Dashboard con potencial solar en tiempo real
- Pronostico por horas con graficos de barras
- Planificador inteligente de electrodomesticos
- Mapa de radiacion solar con Google Maps
- Historial de ahorro diario
- Alertas de clima extremo
- Factor de eficiencia por temperatura
- Agitar para actualizar (Shake to Refresh)
- Ubicacion personalizada con autocompletado
- Configuracion de sistema solar personalizado
- Modo offline con cache de datos
- Notificaciones programadas para hora pico

## Tecnologias Utilizadas

- Flutter 3.x
- Dart 3.x
- Provider (State Management)
- SharedPreferences (Almacenamiento local)
- OpenWeatherMap API (Datos meteorologicos)
- Google Maps API (Mapa de radiacion)
- Geolocator (GPS)
- Sensors Plus (Acelerometro)
- Fl Chart (Graficos)
- Flutter Local Notifications (Notificaciones)
- HTTP (Peticiones API)
